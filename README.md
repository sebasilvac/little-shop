# Little Shop

1. Clonar proyecto
2. instalar dependencias
```yarn install```
3. renombrar archivo ```.env.example``` a ```.env```
4. cambiar variables de entorno
5. levantar base de datos
```docker-compose up -d```
6. levantar proyecto en modo desarrollo
```yarn start:dev```
7. correr seeders
```http://localhost:3000/api/seed```
